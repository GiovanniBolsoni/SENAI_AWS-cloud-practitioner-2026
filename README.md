# AWS Cloud Practitioner — Progresso do Curso

Repositório criado para documentar minha jornada de estudos e atividades práticas do curso **AWS Certified Cloud Practitioner**.

Aqui eu registro os laboratórios que fui realizando na AWS: criação de instâncias EC2, configuração de servidores web (Windows e Linux) e outras práticas do curso, com prints e explicações de cada etapa.

## 🎯 Objetivo

Servir como portfólio de aprendizado — mostrando não só o resultado final de cada atividade, mas o processo até chegar lá.

- **Key Pairs** — geração de chaves para acesso seguro às instâncias
- **Security Groups** — configuração de acesso (implícito na criação das instâncias)
